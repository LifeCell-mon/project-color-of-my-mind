Головна Червоний Помаранчевий Жовтий Зелений Синій Індиго Фіолетовий

Фіолетовий

Креативність, загадковість, магія. Часто для інноваційних та арт‑проєктів.

Тези про колір

  • Поєднує прохолоду синього і енергію червоного.
  • Підходить для преміум‑відчуття з золотими акцентами.
  • У UI — для особливих станів та бейджів.

Поради з поєднання

Ефектно з золотим, білим, темним сірим.

В UI цей колір доречно використовувати для: кнопок, бейджів, виділення посилань. Для фону — знижуй насиченість або обирай найсвітліші відтінки.
