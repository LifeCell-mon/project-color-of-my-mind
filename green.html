Зелений

Природа, баланс, зростання. Добре для еко‑тем, фінансів та здоров’я.

Тези про колір

  • Заспокоює, асоціюється з безпекою та довірою.
  • Темні відтінки — для преміальності, світлі — для свіжості.
  • У UI часто позначає успіх/підтвердження.

Поради з поєднання

Чудово з бежевим, білим, темно‑синім.

В UI цей колір доречно використовувати для: кнопок, бейджів, виділення посилань. Для фону — знижуй насиченість або обирай найсвітліші відтінки.
