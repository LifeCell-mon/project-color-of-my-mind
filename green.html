Зелений

Природа, баланс, зростання. Добре для еко‑тем, фінансів та здоров’я.

Тези про колір

  • Заспокоює, асоціюється з безпекою та довірою.
  • Темні відтінки — для преміальності, світлі — для свіжості.
  • У UI часто позначає успіх/підтвердження.

Поради з поєднання

Природні комбінації: з бежевим для тепла, з білим для свіжості, з темно-синім для надійності, з коричневим для земних тонів.

В UI: ідеальний для кнопок підтвердження, успішних статусів, еко-тематики. Світлі відтінки — для фонів, темні — для тексту.
