Синій

Спокій, надійність, технологічність. Найпопулярніший у корпоративному секторі.

Тези про колір

  • Підвищує довіру та відчуття стабільності.
  • Світлі відтінки — дружні, темні — формальні та серйозні.
  • Часто основа для UI‑тем і посилань.

Поради з поєднання

Спокійні комбінації: з помаранчевим для енергії, з білим для чистоти, з жовтим для оптимізму, з сірим для професійності.

В UI: універсальний для посилань, кнопок навігації, головних елементів інтерфейсу. Темно-синій — для текстів, блакитний — для фонів.
