Жовтий

Сонячність, радість, помітність. Сильний акцент, але у надлишку втомлює.

Тези про колір

  • Викликає відчуття радості та молодості.
  • Покращує видимість попереджувальних елементів.
  • Баланс з нейтральними кольорами обов’язковий.

Поради з поєднання

Сонячні комбінації: з графітовим для серйозності, з синім для гармонії, з фіолетовим для креативності, з білим для легкості.

В UI: ефективний для виділення важливих елементів, попереджень, акцентів. Уникай великих жовтих поверхонь — може втомлювати очі.
